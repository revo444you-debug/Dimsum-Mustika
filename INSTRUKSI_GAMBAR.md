# Instruksi Menambahkan Gambar

Untuk menampilkan foto udang, ayam, dan resep warisan di landing page:

## Langkah-langkah:

1. Buat folder `public/images` jika belum ada
2. Tambahkan 3 foto berikut ke folder tersebut:
   - `udang-segar.jpg` - Foto udang segar berkualitas premium
   - `ayam-premium.jpg` - Foto daging ayam premium
   - `resep-warisan.jpg` - Foto proses pembuatan dimsum atau chef

## Catatan:
- Ukuran gambar yang direkomendasikan: 800x600px atau rasio 4:3
- Format: JPG atau PNG
- Jika gambar tidak ditemukan, akan otomatis menampilkan emoji sebagai fallback

## Lokasi file:
```
public/
  └── images/
      ├── udang-segar.jpg
      ├── ayam-premium.jpg
      └── resep-warisan.jpg
```

Setelah menambahkan gambar, refresh browser untuk melihat perubahan.
