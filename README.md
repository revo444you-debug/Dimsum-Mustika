# Dimsum Landing Page

Landing page bertema Dimsum menggunakan Next.js dan Tailwind CSS.

## Cara Menjalankan

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Buka browser di [http://localhost:3000](http://localhost:3000)

## Fitur

- Hero section dengan call-to-action
- Menu dimsum favorit
- Section tentang kami
- Informasi kontak
- Responsive design
- Styling dengan Tailwind CSS

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
